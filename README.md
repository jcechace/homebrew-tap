# Homebrew Tap

Homebrew formulae for tools by [Jakub Cechacek](https://github.com/jcechace).

## Installation

```bash
brew tap jcechace/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [pbmate](https://github.com/jcechace/pbmate) | TUI companion for Percona Backup for MongoDB |

```bash
brew install pbmate
```

## Updating

```bash
brew update
brew upgrade <formula>
```

## About

Formulae are updated automatically on each release. For issues and source code, see the linked repositories above.
